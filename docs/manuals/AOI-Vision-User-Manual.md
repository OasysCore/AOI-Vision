## 一、系统概述

AOI-Vision（自动光学检测系统）是潤芯國際 OASYS CORE 自主研发的工业视觉检测平台，基于 OasysCoreCV 核心引擎，提供缺陷检测、精密量测、SPC 统计过程控制、设备遥测等完整功能。

### 系统架构

```
┌─────────────────┐     ┌──────────────────┐     ┌─────────────────┐
│   浏览器 UI       │ ←→ │   FastAPI 业务层   │ ←→ │  OasysCoreCV 引擎 │
│  (HTML/CSS/JS)   │     │  (认证/权限/API)   │     │  (17 种检测模块)  │
└─────────────────┘     └──────────────────┘     └─────────────────┘
                                 ↓
                         ┌──────────────────┐
                         │   HaaS506-HD3    │
                         │   (4G 设备遥测)   │
                         └──────────────────┘
```

### 技术指标

| 指标 | 参数 |
|------|------|
| 检测模式 | 17 种（模板比对/特征匹配/频域分析/立体视觉等） |
| 量测精度 | 亚像素级（μm 级） |
| 相机支持 | USB UVC / IP Camera / 模拟相机 |
| 设备通信 | MQTT (HaaS506-HD3) / Mock 模拟 |
| 数据库 | SQLite（本地开发）/ PostgreSQL（生产） |

## 三、功能操作

### 3.1 检测操作台（默认首页）

登录后进入检测操作台，左半部分为实时相机画面，右半部分为参数设定面板。

**操作步骤：**

1. 点击 **▶ 开启相机** 按钮，启动实时视频流
2. 在右侧面板选择检测参数：
   - **Profile 设定**：选择预存的检测配置方案
   - **检测参数**：选择检测模式（模板比对/轮廓分析/颜色阈值）
   - **设备**：选择设备网关（Mock/HaaS506-HD3）
3. 点击 **🔍 拍照并检测** 执行检测
4. 下方结果面板显示检测结果

![检测操作台]
> **图片标识: manual-operation** — 检测操作台主界面，左侧相机画面 + 右侧参数面板 + 底部检测结果。

**检测结果解读：**

| 字段 | 说明 |
|------|------|
| 判定 | PASS（合格）/ FAIL（不合格） |
| 缺陷数 | 检测到的缺陷总数量 |
| 置信度 | 检测可信度（0-100%） |
| 缺陷详情 | 每个缺陷的类型、严重度、面积、位置 |

![检测结果]
> **图片标识: manual-result** — 检测结果面板，显示判定/缺陷数/置信度/缺陷详情表。

### 3.2 缺陷规则管理

点击顶部导航栏 **🔍 缺陷规则** 标签。

**功能：**
- **创建规则**：输入名称、选择检测类型（模板比对/轮廓分析/颜色阈值）、设置阈值，点击 **+ 创建**
- **查看规则列表**：显示所有已创建的规则及其状态
- **执行检测**：点击某条规则的 **检测** 按钮立即执行
- **删除规则**：点击 **删除** 按钮移除规则

![缺陷规则]
> **图片标识: manual-defects** — 缺陷规则管理页面，显示创建表单 + 规则列表。

### 3.3 量测校准

点击顶部导航栏 **📐 量测校准** 标签。

**标定操作：**
1. 输入名称、像素距离、实际距离（mm）
2. 点击 **计算** 按钮
3. 系统自动计算 `pixels/mm` 标定值

**页面包含：**
- 标定列表：历史标定记录
- 量测记录：历次量测数据

![量测校准]
> **图片标识: manual-measure** — 量测校准页面，显示标定计算 + 列表 + 量测记录。

### 3.4 SPC 报表

点击顶部导航栏 **📈 SPC 报表** 标签。

**显示内容：**
| 指标 | 说明 |
|------|------|
| CPK | 过程能力指数 |
| PPK | 过程性能指数 |
| σ (Sigma) | 标准差 |
| 良品率 | 合格品百分比 |

**趋势表**：按日期显示缺陷数和判定结果。

**PDF 报告**：点击 **📄 下载PDF报告** 导出统计报表。

![SPC报表]
> **图片标识: manual-spc** — SPC 统计报表页面，显示 CPK/PPK/σ/良率卡片 + 趋势表。

### 3.5 设备监控

点击顶部导航栏 **📡 设备监控** 标签。

**显示内容：**
- 设备在线状态
- 4G 信号强度
- 温度
- 运行时长
- **继电器控制**：远程开关（继电器1/继电器2）

![设备监控]

### 3.6 审计日志

点击顶部导航栏 **📋 审计日志** 标签。

**功能：**
- 按用户筛选：输入用户名查询

## 五、权限体系

### 5.1 权限模型

```
权限组 (Permission) ←──→ 账号组 (AccountGroup) ←──→ 用户 (User)
    多对多                      多对多
```

### 5.2 预置权限组

| 权限组 | 适用范围 | 权限范围 |
|--------|---------|---------|
| 全权限管理 | 系统管理员 | inspection/measurement/defects/reports/device/admin/users/groups/permissions → manage |
| 质检员权限 | 质量主管 | inspection/measurement/defects → manage |
| 操作员权限 | 产线操作 | inspection → write，其余 read |
| 只读权限 | 访客/客户 | 全部 → read |

### 5.3 预置账号组

| 账号组 | 关联权限组 |
|--------|----------|
| 管理员组 | 全权限管理 |
| 质检员组 | 质检员权限 |
| 操作员组 | 操作员权限 |
| 访客组 | 只读权限 |

## 六、常见问题

**Q: 相机无法开启？**
A: 检查相机是否连接，确认 `.env` 中 `CAMERA_TYPE=mock`（开发）或 `CAMERA_TYPE=uvc`（生产）。

**Q: 检测结果全是 PASS？**
A: Mock 模式下使用模拟数据。连接真实相机后使用实际图像检测。

**Q: 如何切换生产数据库？**
A: 修改 `.env` 中 `DATABASE_URL` 为 PostgreSQL 连接串：
```
DATABASE_URL=postgresql+asyncpg://user:password@host:5432/aoi_vision
```
