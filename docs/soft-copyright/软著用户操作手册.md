## 一、软件概述

### 1.1 软件简介

AOI-Vision 是潤芯國際 OASYS CORE 自主研发的工业视觉检测平台，基于 OasysCoreCV 核心引擎，提供缺陷检测、精密量测、SPC 统计过程控制、设备遥测等完整功能。系统采用三层架构（UI 展示层 / FastAPI 业务层 / OasysCoreCV 核心层），支持 17 种检测模块按需开关，通过多对多 RBAC 权限系统实现精细化访问控制。

### 1.2 运行环境

| 类别 | 要求 |
|------|------|
| 操作系统 | macOS 14+, Ubuntu 22.04+, Windows Server 2022 (Docker) |
| CPU | x86_64 双核以上 |
| 内存 | 4 GB+（推荐 8 GB） |
| 浏览器 | Chrome 90+, Firefox 90+, Edge 90+ |
| 数据库 | SQLite 3.35+（开发）/ PostgreSQL 14+（生产） |
| Python | 3.11+ |

## 三、部署流程

### 3.1 环境准备

```bash
# 1. 安装 Python 3.11+
python3 --version

# 2. 获取源码
git clone https://github.com/OasysCore/AOI-Vision.git
cd AOI-Vision/backend

# 3. 创建虚拟环境
python3.11 -m venv .venv
source .venv/bin/activate

# 4. 安装依赖
pip install -r requirements.txt
```

### 3.2 配置文件

创建 `.env` 文件：

```ini
DATABASE_URL=sqlite+aiosqlite:///./aoi_vision.db
JWT_SECRET_KEY=your-p...n
CAMERA_TYPE=mock
DEVICE_GATEWAY_TYPE=mock
```

### 3.3 启动服务

```bash
cd backend
.venv/bin/uvicorn app.main:app --host 0.0.0.0 --port 8000
```

### 3.4 验证部署

```bash
curl http://localhost:8000/health
# 预期输出: {"status":"ok","version":"0.1.0"}
```

浏览器访问 `http://localhost:8000`，使用默认账号 `admin/admin123` 登录。

## 四、主要功能操作

### 4.1 登录系统

1. 打开浏览器访问 `http://localhost:8000`
2. 输入用户名 `admin`，密码 `admin123`
3. 点击 **登录** 按钮
4. 进入检测操作台主界面

> **图片标识: copyright-login** — 登录页面截图

### 4.2 检测操作台（核心功能）

1. 点击 **▶ 开启相机** 启动实时视频流
2. 在右侧面板选择检测模式（模板比对/轮廓分析/颜色阈值）
3. 配置检测参数（阈值/最小面积等）
4. 点击 **🔍 拍照并检测** 执行检测
5. 下方结果面板显示：判定（PASS/FAIL）、缺陷数、置信度、缺陷详情表

> **图片标识: copyright-operation** — 检测操作台主界面

### 4.3 缺陷规则管理

1. 点击顶部导航 **🔍 缺陷规则**
2. 输入规则名称、选择类型、设置阈值
3. 点击 **+ 创建** 保存规则
4. 点击规则旁的 **检测** 按钮执行

> **图片标识: copyright-defects** — 缺陷规则管理页面

### 4.4 量测校准

1. 点击顶部导航 **📐 量测校准**
2. 输入像素距离和实际距离
3. 点击 **计算** 获得 pixels/mm 标定值

> **图片标识: copyright-measure** — 量测校准页面

### 4.5 SPC 统计报表

1. 点击顶部导航 **📈 SPC 报表**
2. 查看 CPK/PPK/σ/良品率 统计卡片
3. 查看日期-缺陷数-判定趋势表
4. 点击 **📄 下载PDF报告** 导出

> **图片标识: copyright-spc** — SPC 报表页面

### 4.6 系统管理

1. 点击 **⚙ 系统管理** → 包含 5 个子标签页
2. **👤 用户** — 创建/禁用用户，分配账号组
3. **👥 账号组** — 创建组并关联权限
4. **🔑 权限组** — 查看 4 个预置权限组
5. **📝 参数枚举** — 管理缺陷类型/严重度等枚举值
6. **🧩 功能模块** — 17 种检测模块的启用/禁用开关

> **图片标识: copyright-admin** — 系统管理页面

### 4.7 设备监控

1. 点击顶部导航 **📡 设备监控**
2. 查看设备在线状态、4G 信号、温度、运行时长
3. 远程控制继电器开关

> **图片标识: copyright-device** — 设备监控页面

### 4.8 审计日志

1. 点击顶部导航 **📋 审计日志**
2. 按用户名或操作类型筛选
3. 查看操作审计记录

> **图片标识: copyright-logs** — 审计日志页面
