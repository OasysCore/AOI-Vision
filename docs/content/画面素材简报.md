**系统录屏方法（macOS）：**
```bash
# QuickTime Player → 新建屏幕录制 → 选择Chrome窗口（http://127.0.0.1:8000）
# 或命令行：
screencapture -v -C screen_recording.mov
```

**素材来源优先级：**
1. 系统录屏/截图（AOI-Vision实际运行画面）← 最优先
2. 实物实拍（PCB/零件/产线环境）← 真实感
3. 后期合成/动画（原理示意/数据对比卡）← 专业感
4. 素材库（质检员工位/工厂全景）← 补充镜头
