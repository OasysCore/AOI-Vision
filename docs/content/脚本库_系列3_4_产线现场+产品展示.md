## E07：某电子厂质检员的一天

**BGM**: vlog日常感

|------|------|----------|

## E09：客户问：你的机器会不会漏检？

**时长**: 30-40秒
**BGM**: 前半段反问感→后半段证据硬核

### 脚本

| 时间 | 画面 | 旁白/字幕 |
|------|------|----------|
| 0-3s | 客户对话场景（文字形式）：「你这机器会不会漏检？」 | "客户总会问一个问题——" |
| 3-6s | 摇头特写 | "会不会漏检？" |
| 6-10s | 拿出一块故意制造了3处缺陷的样本，放在检测台上 | "来，我现场给你做一块有问题的" |
| 10-16s | OasysCoreCV 检测运行，3处缺陷逐一标出→FAIL | "三个缺陷，全部检出" |
| 16-20s | 换个角度放，再测一次→仍然检出 | "换个角度，再来一次——" |
| 20-24s | 换个光照条件，再测→仍然检出 | "换个光照——还是一样的结果" |
| 24-28s | 3次检测结果并列对比：一致检出3个缺陷 | "三次检测，百分百一致" |
| 28-32s | 结尾 LOGO + 文字「来，测给你看」 | "漏检？来，我测给你看" |
