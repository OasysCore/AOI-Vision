## 一、即梦能做什么

| 功能 | 用途 | AOI视频对应 |
|------|------|------------|
| **AI数字人出镜** | 用你的数字人形象+声音，讲脚本 | 主持人讲解 |
| **文生视频** | 输入文字描述 → 生成视频片段 | 工厂产线/检测画面/科技感场景 |
| **图生视频** | 上传图片 → AI让图片动起来 | 产品特写动画 |

| **视频编辑** | 拼接/字幕/转场/配乐 | 后期合成 |

## 二、完整操作流程（7步）

### 第1步：准备工作

```
1. 下载即梦App（iOS App Store / Android应用商店搜索"即梦"）
   或访问网页版：https://jimeng.jianying.com
2. 用抖音账号登录
3. 确认你的AI数字人已经创建好（路径：我的 → 数字人 → 创建/管理）
```

### 第2步：录制数字人出镜视频

```
操作路径：首页 → 数字人 → 选择你的数字人形象
1. 选择背景（纯色/模糊/自定义图）
2. 选择声音（你的克隆声音 / 系统音色）
3. 输入脚本文字 → 点击生成
4. 等待1-3分钟 → 下载视频
```

**关键技巧：**
- 每段脚本控制在15-20秒，方便后期拼接
- 开头、中间转换、结尾各录一段
- 声音用自己克隆的更自然

### 第3步：生成 B-roll 画面

B-roll是数字人说话时穿插播放的画面。用**文生视频**生成：

```
操作路径：首页 → 文生视频
1. 输入画面描述（英文效果更好，见下方prompt库）
2. 选择视频风格：写实 / 科幻 / 工业
3. 选择时长：4秒或8秒
4. 点击生成 → 每次4条候选 → 选最好的一条
5. 下载备用
```

### 第4步：生成封面图和对比图

```
操作路径：首页 → 文生图
1. 输入描述 → 生成
2. 选最满意的一张
3. 用"图片编辑"加文字标注
4. 导出作为视频封面
```

### 第5步：剪辑合成

```
工具：剪映（抖音官方剪辑工具，手机版/专业版都行）
步骤：
1. 导入所有素材（数字人视频 + B-roll视频 + 对比图）
2. 时间线排列：
   - 00:00-00:03  数字人开场
   - 00:03-00:08  B-roll画面（数字人画外音继续）
   - 00:08-00:12  数字人转场
   - 00:12-00:20  B-roll画面
   - 00:20-00:25  数字人结尾
3. 添加字幕：剪映"智能字幕"→自动识别语音→一键生成字幕
4. 添加BGM：剪映音频库 → 搜索"科技""工业""紧张"
5. 导出：1080p 30fps 竖屏9:16
```

### 第6步：发布

```
抖音发布设置：
- 封面：用第4步生成的对比图
- 标题：按脚本中的标题写
- 标签：#工业视觉 #自动化检测 #质量管控 #AI检测
- 位置：不填或填公司位置
- 发布时间：周一三五 中午12:00 或 晚上19:00-21:00
```

### 第7步：TikTok 海外版同步

```
TikTok发布：
- 把字幕改成英文（剪映导出时选英文字幕）
- 标签改用：#IndustrialVision #QualityControl #AOI #MachineVision
- 发布时间：按美国/东南亚时区调整
```

## 三、即梦 Prompt 库

### 数字人出镜（直接用你的数字人+文字脚本）

直接在即梦数字人页面输入脚本文字即可，不需要额外prompt。

### 文生视频 B-roll Prompts

**工厂产线场景：**
```
A modern electronics factory production line with conveyor belt moving circuit boards. 
Clean room environment, bright white industrial lighting. 
Camera slowly tracking along the assembly line. Professional documentary style. 4K.
```

**检测系统运行：**
```
Close-up of an automated optical inspection machine scanning a green PCB circuit board. 
Blue laser scanning light moving across the board. 
Digital HUD overlay showing inspection data and PASS indicators. 
High-tech industrial aesthetic, dark environment, dramatic lighting.
```

**PCB缺陷特写：**
```
Extreme close-up macro shot of a circuit board solder joint with a tiny crack. 
The crack becomes highlighted with red glow. 
Microscope perspective, shallow depth of field, dark background. 
Cinematic lighting.
```

**数据面板展示：**
```
Futuristic transparent holographic dashboard floating in dark control room. 
Displaying quality control statistics: yield rate, defect count, CPK charts. 
Blue and cyan neon glow. Sci-fi but professional aesthetic.
```

**对比分屏：**
```
Split screen. Left side: a tired worker using magnifying glass in warm yellow light. 
Right side: a sleek robotic inspection system with blue LED scanning. 
Both sides examining the same product. Dramatic contrast in lighting and mood.
```

**零件计数：**
```
Thousands of tiny electronic components scattered on a black inspection surface. 
Digital counting overlay appearing on top, numbers rapidly increasing. 
Top-down camera angle. Clean professional look.
```
